```html Angular UI Testing Frameworks Cheat Sheet
Angular UI Testing Frameworks Cheat Sheet

Protractor

Installation: npm install -g protractor

Run tests: protractor conf.js

Update WebDriver: webdriver-manager update

Karma

Installation: npm install karma --save-dev

Run tests: karma start karma.conf.js

Run single test: fit('test name')

Jasmine

Installation: npm install -g jasmine

Run tests: jasmine

Describe block: describe("description", function(){...})

Cypress

Installation: npm install cypress --save-dev

Run tests: npx cypress open

Cypress command: cy.visit('url')

```
