GitHub Workflows Cheat Sheet

Common commands
git clone [url]
git status
git add .
git commit -m "message"
git push
Workflow syntax
on:
  push:
    branches:
      - 'main'
jobs:
  build:
    runs-on: ubuntu-latest
steps:
  - uses: actions/checkout@v2
Best practices

Use clear and descriptive commit messages.

Keep your workflows as simple as possible.

Use actions created by others when possible.
