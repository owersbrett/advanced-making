```html iOS Deployment Cheat Sheet
iOS Deployment Cheat Sheet

Xcode

  • Build: Cmd + B
  • Run: Cmd + R
  • Test: Cmd + U
  • Archive: Cmd + Shift + A
  • Validate: Product -> Validate App
  • Distribute: Product -> Distribute App

iTunes Connect

  • Create new app: My Apps -> +
  • Create new version: App -> + Version
  • Submit for review: Save -> Submit for Review

Certificates

  • Create: Apple Developer -> Certificates -> +
  • Download: Download
  • Install: Double click downloaded file

Provisioning Profiles

  • Create: Apple Developer -> Provisioning Profiles -> +
  • Download: Download
  • Install: Double click downloaded file
```
