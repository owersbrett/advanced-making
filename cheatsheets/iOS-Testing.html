iOS Testing Cheat Sheet

Common Commands

xcodebuild test -project MyApp.xcodeproj -scheme MyApp xcodebuild test -workspace MyApp.xcworkspace -scheme MyApp xcodebuild test -destination 'platform=iOS Simulator,name=iPhone 12'

Syntax

XCTAssert(condition, "message") XCTAssertTrue(condition, "message") XCTAssertFalse(condition, "message") XCTAssertEqual(a, b, "message") XCTAssertNotEqual(a, b, "message")

Best Practices

1. Test all business logic.

2. Keep tests independent from one another.

3. Use mock objects to simulate complex data scenarios.

4. Test asynchronous code with expectations.

5. Use code coverage to identify untested code.
