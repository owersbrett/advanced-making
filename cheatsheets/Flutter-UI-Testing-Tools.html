Flutter UI Testing Tools Cheat Sheet

Flutter Driver

flutter drive --target=test_driver/app.dart - Runs instrumented tests on device/emulator

find.byValueKey('key') - Finds a widget by its value key

driver.tap(buttonFinder) - Simulates a tap on the buttonFinder widget

Widget Testing

flutter test test/widget_test.dart - Runs widget tests

testWidgets('description', (WidgetTester tester) async {}) - Defines a widget test

await tester.pumpWidget(MyApp()) - Renders the MyApp widget

Integration Testing

flutter test integration_test/app_test.dart - Runs integration tests

setUpAll(() async {}) - Runs before all tests

tearDownAll(() async {}) - Runs after all tests
