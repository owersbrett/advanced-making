```html Android Testing Cheat Sheet
Android Testing Cheat Sheet

Common Commands

  • ./gradlew test: Run unit tests
  • ./gradlew connectedAndroidTest: Run instrumented tests
  • ./gradlew jacocoTestReport: Generate code coverage report

Syntax

  • @Test: Marks a method as a test method
  • @Before: Runs before each test
  • @After: Runs after each test
  • @Mock: Creates a mock implementation

Best Practices

  • Write small, focused, single-purpose tests
  • Separate UI tests from unit tests
  • Mock out dependencies in unit tests
  • Use Espresso for UI tests
```
