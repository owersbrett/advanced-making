```html Android UI Testing Cheat Sheet
Android UI Testing Cheat Sheet

Common Commands

  • adb shell am instrument -w [test package]/androidx.test.runner.AndroidJUnitRunner
  • gradlew connectedAndroidTest
  • gradlew test

Espresso Syntax

  • onView(withId(R.id.view_id)).perform(click())
  • onView(withId(R.id.view_id)).check(matches(isDisplayed()))
  • onView(withText("text")).perform(typeText("input"))

Best Practices

  • Always clean up after your tests
  • Use Test Doubles for external dependencies
  • Make your tests hermetic
  • Run your tests on real devices whenever possible
```
