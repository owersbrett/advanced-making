```html Android Development Cheat Sheet

Common Commands

  • ./gradlew build - Build your app
  • ./gradlew test - Run tests
  • ./gradlew installDebug - Install debug build
  • adb logcat - View log output

Syntax

  • val - Define a read-only property
  • var - Define a mutable property
  • lateinit var - Late initialization
  • ::class.java - Get Java class of an object

Best Practices

  • Use MVVM architecture
  • Use LiveData for reactive UI
  • Avoid memory leaks by careful use of Context
  • Use ProGuard to shrink your APK size
```
