```html iOS Localization Cheat Sheet
iOS Localization Cheat Sheet

Commands

Extract strings:

genstrings -o en.lproj *.m

Update strings:

ibtool --generate-strings-file file.strings file.xib

Syntax

Localized String:

NSLocalizedString(@"key", @"comment")

Custom table:

NSLocalizedStringFromTable(@"key", @"table", @"comment")

Best Practices

1. Always provide meaningful comments.

2. Keep keys unique.

3. Use format specifiers for dynamic content.

4. Test localizations regularly.

```
