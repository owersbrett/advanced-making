Angular CLI

  • ng new my-app: Creates a new Angular project.
  • ng serve: Starts the development server.
  • ng generate component my-component: Generates a new component.
  • ng build: Compiles the application into an output directory.
  • ng test: Runs unit tests in a project.

Angular Syntax

  • {{data}}: Interpolation
  • [property]="data": Property binding
  • (event)="expression": Event binding
  • [(ngModel)]="data": Two-way data binding
  • *ngFor="let item of items": Structural directive
  • *ngIf="condition": Conditional rendering

Best Practices

  • Use Angular CLI for boilerplate code
  • Follow Angular's style guide
  • Keep components small and focused
  • Use services for data sharing
  • Prefer Observables over Promises
