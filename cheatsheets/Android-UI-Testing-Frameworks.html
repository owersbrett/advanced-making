```html Android UI Testing Frameworks Cheat Sheet

Espresso

onView(withId(R.id.my_view))
.perform(click());
onView(withText("Hello"))
.check(matches(isDisplayed()));

UI Automator

new UiScrollable(new UiSelector()
.scrollable(true).instance(0))
.scrollIntoView(new UiSelector()
.textContains("@string/my_string").instance(0));
UiObject checkBox = mDevice.findObject(new UiSelector().checkable(true));

Robolectric

ActivityScenario.launch(LoginActivity.class);
Robolectric.setupActivity(MyActivity.class);

Best Practices

  • Use Espresso for UI tests within an app.
  • Use UI Automator for testing between apps and system features.
  • Use Robolectric for fast running tests that do not need Android runtime.
  • Test early, test often: Use TDD/BDD principles.
  • Always run the full test suite before a release.
```
