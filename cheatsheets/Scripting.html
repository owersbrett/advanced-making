```html Scripting Cheat Sheet

Common Commands

echo: Outputs strings or variables

printf: Formats and outputs data

read: Reads a line from standard input

exit: Exits the shell or script

Syntax

Variables: VARNAME="value"

Loops: for VAR in LIST; do COMMANDS; done

If statements: if [ CONDITION ]; then COMMANDS; fi

Functions: function NAME() { COMMANDS; }

Best Practices

Quoting: Always quote variable references

Indentation: Consistently indent for readability

Error Handling: Always check command exit status

Comments: Use comments to explain complex code

```
