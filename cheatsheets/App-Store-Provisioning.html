App Store Provisioning Cheat Sheet


# 1. Create App ID
- appstoreconnect.apple.com > Certificates, IDs & Profiles > Identifiers > '+' > App IDs

# 2. Create Provisioning Profile
- Certificates, IDs & Profiles > Profiles > '+' > iOS App Development (or relevant)

# 3. Download & Install Provisioning Profile
- Double click or Xcode > Preferences > Accounts > Download Manual Profiles

# 4. Code Signing
- Xcode > Target > Signing & Capabilities > "Team" select your team

# 5. Build Setting > Code Signing Identity
- "Apple Development" for debug
- "Apple Distribution" for release

# 6. Archive & Validate
- Product > Archive > Validate App

# 7. Distribute App
- Product > Archive > Distribute App > Upload
